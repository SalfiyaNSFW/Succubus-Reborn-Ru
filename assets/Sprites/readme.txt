В данной каталоге можно заменить изображения карточек игры,
за исключением анимированных карт. (В основном это карты суккубов)

Игра поддерживает изображение в формате PNG, а его размеры не должны превышать 630x880 пикселей (включая рамку карточки)

Название файла должно совпадать с идентификатором карты.
ID карты можно посмотреть в игре при редактировании колоды.
Например: GD006 - Это карта миньона (адская гончая / Farm-Bred Hucow)
